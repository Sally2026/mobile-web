Takeaways from the Course

I learned a lot from this course. I learned how to use CSS Grid and Flexbox to create responsive layouts. I also learned how to use SCSS to write cleaner and more maintainable CSS. I learned how to use Bootstrap to quickly create responsive designs. I also learned how to use the srcset attribute to serve different images for different screen sizes. I learned how to use the video and audio elements to embed media in my website. Overall, I feel more confident in my web design and development skills.
